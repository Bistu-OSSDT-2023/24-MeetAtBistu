
# 项目名称

Meet At Bistu(MAB)  
本项目是基于微信小程序的交友软件，旨在为北京信息科技大学的广大同学提供一个结交新朋友扩大自己的社交圈的平台。我们经过讨论发现以我们现有的条件从零实现这个项目困难重重，所以我们从Github上选择了一个开源项目，并在原基础上继续完善此项目。

## 功能特性

- 允许用户发布活动，所有用户都可以看见其他用户发布的活动并选择参加活动。
- 发布瞬间，分享你生活的点点滴滴。（类似于QQ空间或朋友圈的功能）
